
# JS Algorithms Practice

A collection of JavaScript algorithm problems for hands-on training.

## Junior Training

- 39 essential tasks: strings, arrays, hash maps/objects, logic/math, recursion
- Ready-to-use templates and clear problem descriptions
- See the [Junior Training folder](algorithms-junior-training) and its [README](algorithms-junior-training/README.md) for details

## How to Use

1. Open any `.js` file in the junior training folder
2. Read the problem and implement your solution
3. Test with Node.js
