# JavaScript Algorithms - Junior Training

It is a collection of algorithm practice problems organized by category.

## Structure

### Strings
- `reverseString.js` - Reverse a string
- `palindrome.js` - Check if a string is a palindrome
- `countCharacters.js` - Count characters in a string
- `removeSpaces.js` - Remove all spaces from a string
- `capitalizeEachWord.js` - Capitalize the first letter of each word

- `countVowels.js` - Count vowels in a string

### Arrays
- `max.js` - Find maximum value
- `min.js` - Find minimum value
- `reverseArray.js` - Reverse an array
- `removeDuplicates.js` - Remove duplicate values
- `twoSum.js` - Find two indices that sum to target
- `mergeTwoArrays.js` - Merge two arrays
- `mergeSortedArrays.js` - Merge two sorted arrays
- `sumOfElements.js` - Calculate sum of elements
- `findSecondLargest.js` - Find second largest value
- `findIndexOfElement.js` - Find index of element
- `rotateArray.js` - Rotate array by N steps

### Hash Maps / Objects
- `countOccurrences.js` - Count occurrences in array
- `firstRepeating.js` - Find first repeating element
- `firstNonRepeating.js` - Find first non-repeating character
- `groupElements.js` - Group elements by property

### Logic / Math
- `fizzbuzz.js` - Classic FizzBuzz problem
- `isPrime.js` - Check if number is prime
- `factorialIterative.js` - Calculate factorial (iterative)
- `fibonacciIterative.js` - Generate Fibonacci sequence (iterative)
- `countEvenOdd.js` - Count even/odd numbers
- `powerOfTwo.js` - Check if number is power of two
- `flattenArray.js` - Flatten nested array
- `removeFalsy.js` - Remove falsy values from array
- `chunkArray.js` - Split array into chunks
- `deepClone.js` - Deep clone an object
- `compareObjects.js` - Compare two objects (shallow)

### Recursion
- `factorialRecursive.js` - Calculate factorial (recursive)
- `sumArrayRecursive.js` - Sum array elements (recursive)
- `countDigitsRecursive.js` - Count digits (recursive)

## How to Use:

1. Navigate to any category folder
2. Open a `.js` file
3. Read the problem description in the comments
4. Implement your solution
5. Test your solution by running the file with Node.js
